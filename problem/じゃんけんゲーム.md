# じゃんけんゲーム

以下の仕様を満たすコードを実装してください．

## 説明


```
python janken.py
<< 0: グー, 1: チョキ, 2:パー
<< じゃんけん...
>> 0
<< あなた: グー, CPU: チョキ
<< あなたの勝ちです

python janken.py
<< 0: グー, 1: チョキ, 2:パー
<< じゃんけん...
>> 1
<< あなた: チョキ, CPU: グー
<< CPUの勝ちです

python janken.py
<< 0: グー, 1: チョキ, 2:パー
<< じゃんけん...
>> 3
<< 0, 1, 2のいずれかの数字を入力してください
```
